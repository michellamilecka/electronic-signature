◆ generate_public_private_RSA_keys()

key_generator.generate_public_private_RSA_keys ( )

Generuje parę kluczy RSA (publiczny i prywatny) o długości 4096 bitów.

Generujemy klucze RSA prywatny i publiczny o długości 4096 bitów z użyciem funkcji random_bytes, która zapewnia, że klucze są losowe. Eksportuje klucz publiczny i prywatny. Zapisuje klucz publiczny do pliku 'public_key.pem'.

Returns
Wygenerowane klucze: publiczny i prywatny.

◆ on_generate_keys()

key_generator.on_generate_keys ( )

Reaguje na kliknięcie przycisku generowania kluczy.

Zapewnia, że PIN jest 4 cyforwy, generuje klucze RSA oraz szyfruje klucz prywatny za pomocą wprowadzonego PIN-u. Informuje użytkownika o aktualnym stanie pracy (szyforwanie kluczy, sukces, błąd).

◆ private_key_encryption()

key_generator.private_key_encryption ( private_key,
pin )

Szyfruje klucz prywatny za pomocą podanego PIN-u przez użytkownika oraz algorytmu AES.

Kod PIN podany przez użytkownika jest zamieniony na bajty. Tworzy klucz AES poprzez hashowanie tego PIN-u za pomocą SHA-256, zwraca 256 bitów. Tworzy obiekt do szyfrowania AES w trybie Cipher Block Chaining. Tworzy wektor inicjujący, który jest potrzebnt do szyfrowania. Szyfruje klucz prywatny, dbając o to, żeby dane miały długość wielokrotności rozmiaru bloku AES. Sprawdza czy jest podlaczony pendrive. Zapisuje zaszyfrowany klucz prywatny do pliku 'private_key_encrypted.pem' na pendrive.

Parameters
private_keyKlucz prywatny do zaszyfrowania.
pinPodany przez użytkownika 4-cyfrowy kod PIN.
